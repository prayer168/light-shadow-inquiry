## 4. 正式版

【數位教材分享】光影遊戲的探究與實作

本教材針對國小自然「光與影」單元設計，以探究與實作為主軸，引導學生透過互動操作建立科學概念。

教材內容包含：
一、互動實驗室：學生可調整光源高度、遠近與物體材質，即時觀察影子大小、長短與深淺的變化，理解變因與結果之關係。
二、分段動畫：逐步演示影子形成歷程，協助學生掌握「光線受阻形成暗區」之原理。
三、情境應用卡：提供八則生活情境，培養學生判斷與控制變因的能力。
四、闖關測驗：共十題，回顧各單元探究歷程，即時回饋並提供解析。

本教材可於電腦、平板及手機開啟使用，免費提供教學現場與親子共學運用，歡迎教師同好參考與分享。
