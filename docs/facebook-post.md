## 長版

🔦✨《光影遊戲的探究與實作》互動教材上線囉！

「為什麼影子有時候大、有時候小？」「為什麼影子會跑到另一邊？」
這份教材讓孩子不只是看答案，而是親手實驗、自己找出規則 👀

🧪 **互動實驗室**：拖動滑桿調整光源高度、遠近和物體材質，螢幕上的影子即時變大、變長、變淡，因果關係一目了然。

🎬 **分段動畫**：一步一步看影子怎麼形成，理解「光被擋住 → 暗區 → 影子」的順序。

🃏 **情境應用卡**：8 種生活情境（影子太小、跑到右邊、形狀不像…），讓孩子判斷「該改變什麼變因」。

✅ **闖關測驗**：10 題回顧前面每個關卡的探究歷程，邊玩邊檢核學會了沒。
