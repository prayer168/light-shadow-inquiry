# 教案建議

## 課程目標

- 能描述影子形成的必要條件。
- 能透過操作光源、物體與承接面，觀察影子大小、位置、長短與深淺。
- 能設計一次只改變一個變因的公平比較。
- 能用觀察證據解釋光影變化。

## 活動流程

### 引起動機

展示手電筒、紙偶與白紙，請學生預測把紙偶移近手電筒時影子會怎樣。

### 探究操作

學生分組使用互動分頁或實體器材，依序改變光源高度、物體距離與材質。每次只改變一個條件，記錄影子的變化。

### 分享討論

請各組用句型說明：「如果我改變＿＿，那麼影子的＿＿會改變，因為＿＿。」

### 應用挑戰

完成情境卡，設計一個「讓影子變大、變淡或變長」的小遊戲，並說明控制變因與觀察變因。

### 評量

使用測驗分頁進行形成性評量。教師可觀察學生是否能用光源、物體、影子三者關係解釋答案。

## 安全提醒

不要直視強光，不要用手電筒照射同學眼睛。若使用手機燈，請降低亮度並控制照射時間。
