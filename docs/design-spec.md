# 設計規格

## 對象

國小自然科學課堂，可供中高年級學生操作，也可作為師培生設計光影探究活動的示範。

## 學習流程

1. 任務：設定學生為光影實驗室研究員。
2. 探索：建立光源、物體、影子的基本概念。
3. 互動：調整光源高度、物體距離與材質。
4. 動畫：分段觀察影子形成順序。
5. 應用：用情境卡判斷該改變哪個變因。
6. 測驗：10 題形成性評量。
7. 資源：保留來源、影片與課堂材料建議。

## 互動設計

- 滑桿控制光源高度與物體距離。
- 單選控制物體材質：不透明、半透明、透明。
- SVG 影子會同步改變大小、位置與深淺。
- 情境卡點擊後顯示建議，並以影子變形提示概念。
- 測驗作答後立即鎖定並顯示詳解。

## 可近用性

- 使用語意化標題與按鈕。
- SVG 提供 title 與 desc。
- 保留 `prefers-reduced-motion`。
- 小螢幕改為單欄或雙欄，避免文字溢出。
