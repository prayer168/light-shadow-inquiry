# 光影遊戲的探究與實作

國小自然科學互動教材，主題取自 CIRN「光影遊戲的探究與實作」。教材以七個分頁組成：任務、探索、互動、動畫、應用、測驗、資源。

## 使用方式

直接開啟 `index.html` 即可使用。教材不需要安裝套件，也不需要啟動伺服器。

## 教學重點

- 影子形成需要光源、物體與承接面。
- 光源、物體與螢幕的距離會影響影子大小。
- 光源角度會影響影子方向與長短。
- 材質透明度會影響影子深淺。
- 公平比較時一次只改變一個變因。

## 檔案

- `index.html`：教材頁面與七分頁結構。
- `styles.css`：版面、RWD 與互動視覺。
- `script.js`：分頁、模擬、動畫、情境卡與測驗邏輯。
- `data/content.js`：應用情境卡內容。
- `data/quiz.js`：10 題測驗與詳解。
- `data/resources.js`：來源與延伸資源。
- `docs/`：教學設計、參考資料與測試紀錄。
